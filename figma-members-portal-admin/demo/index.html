Figma Members Portal + Admin – Umsetzung in 5 Tagen

Diese Seite ist eine Preview/Plan-Ansicht für den Bid. Sie zeigt Vorgehen, Scope-Guardrails und Lieferumfang (Portal + Admin + Reporting).

● Deliverable: Plan + Demo-Preview

Öffentliches Portal

Figma-getreue responsive Umsetzung (Desktop/Tablet/Mobile) inkl. Auth & Member-Dashboard.

  • Registrierung/Login/Logout + Passwort-Reset
  • Mitgliederverwaltung (Profil, Status, Basisdaten)
  • Dashboard für eingeloggte Nutzer (Statistiken)

Admin-Bereich (schlank, erweiterbar)

Rollen & Rechte (RBAC) und wartbare Admin-UI.

  • Nutzer bearbeiten (Rolle, Status, Stammdaten)
  • Mitgliedschaften verwalten (Anmeldung, paid-Status, Laufzeit)
  • Auswertungen: Tag/Woche/Monat/Jahr

5‑Tage Plan (kurz)

  • Tag 1: Setup, Figma→Layout, Auth-Grundgerüst
  • Tag 2: Mitgliederbereich + API + Validierung
  • Tag 3: Admin (Users/Memberships) + RBAC
  • Tag 4: Stats Events + Aggregationen + Reports
  • Tag 5: QA/Responsive Polish + Handover
Arbeitsmodus
Täglicher Review-Link + Abnahmekriterien
Qualität
RBAC, saubere Struktur, erweiterbar

Scope-Guardrails (damit 5 Tage halten)

Zahlungsanbieter-Integration nur, wenn Provider/Flow klar ist (sonst Phase 2).

  • Stats werden zu Beginn als KPI-Liste fixiert
  • Figma-Export wird refaktoriert (Komponenten), nicht 1:1 „zusammenkopiert“
