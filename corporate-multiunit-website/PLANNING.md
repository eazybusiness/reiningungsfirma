# Corporate Multi-Unit Website — Project Planning

## Project Overview
Bid for a corporate website for an industrial multi-unit business group (B2B focus).
Client wants a premium, minimalist, trust-building online presence.

## Deliverables
1. **Bid Proposal** (500-700 characters, German) — `bid-proposal.md`
2. **Video Script** — short script for abbreviated video bid — `video-script.md`
3. **Demo Website** — interactive demo showcasing design concept — `demo/`

## Design Direction
- **Enterprise, not luxury**: Navy dominates, white space, gold as accent only
- **Color Palette**:
  - Primary: Navy `#0A1628` / `#1B2A4A`
  - Accent: Gold `#C8A960` / `#D4AF37`
  - Neutral: White `#FFFFFF`, Light gray `#F5F6FA`, Dark gray `#374151`
- **Typography**: Clean sans-serif (Inter or similar)
- **Layout**: Structured grids, generous spacing, clear hierarchy
- **Gold usage**: Thin borders, icon accents, CTA buttons, dividers — never dominant

## Client Requirements
- CMS-capable (WordPress or Headless CMS)
- Mobile-optimized, DSGVO-compliant, SSL, fast loading
- SEO basics, subtle animations
- 5 Business Units with sub-pages
- Budget: up to €1,800 | Timeline: 3-4 weeks
- Language: German (English expandable later)

## Tech Stack (Demo)
- Vite + React
- TailwindCSS
- Framer Motion (subtle animations)
- Lucide React (icons)
